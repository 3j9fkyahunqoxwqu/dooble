2.01 (03/20/2015)
  1. Ability to hide the main window's menu bar.
  2. Remind the user if the SQLite database driver is missing.
  3. Adjusted full-screen support.
  4. Allow hiding of the Desktop and Home buttons.
  5. The Dogpile search engine supports SSL/TLS.
  6. Added the search engine wikibooks.org.
  7. Corrected dooble::saveSettings() with respect to the full screen mode.
  8. Removed hiding of the location tool bar.
  9. Upgraded OpenSSL libraries to version 1.0.2a.
2.00 (03/01/2015)
  1. Qt 5.4.1 release. Source only. Many, many changes. WebKit has been replaced by WebEngine. Please see dooble.qt5.pro for example preparation. WebKit support persists.
  2. Corrected month translations in the History window.
  3. Allow configuration of the main thread's priority.
  4. Avoid real-time storing of splitter states.
  5. Added a Spot-On button to the location widget.
  6. Removed support for ancient versions of GCC.
