Certificate Exceptions

Web sites may raise SSL/TLS certificate errors. Some of these certificate errors may be overridden.

[INSERT CERTIFICATE ERROR EXCEPTION]

Once overridden, the Web site and the certificate error are recorded in the SQLite dooble_certificate_exceptions.db database file and access to the specific Web site is allowed.

Overridden sites are presented in the Certificate Exceptions window. Within this window, exceptions may also be revoked.

File Menus

Dooble offers a traditional menu bar. The menu bar’s visibility may be configured via the Display panel in the Settings window. If the menu bar is hidden, it may be made visible by pressing the F10 key. Some menu options include mnemonics and shortcuts.

Private Windows

When browsing in private windows, Dooble does not save the following data:

  • Cookies

  • History

  • Temporary Files

  • Visited Links

While in private windows, Dooble does save:

  • Downloads

  • Favorites

Sources of Randomness

Dooble requires data streams of random data for a variety of cryptographic algorithms. This section briefly describes the sources of these data streams.

BSD

BSD-like systems utilize the /dev/random device.

Linux

Linux systems utilizes the /dev/urandom device.

Windows

Please read https://msdn.microsoft.com/en-us/library/windows/desktop/aa379942(v=vs.85).aspx.
